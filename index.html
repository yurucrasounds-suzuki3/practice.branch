Branch Practice

ブランチ切り替えの練習用に作ったシンプルな静的サイトです。

使い方

  1. このリポジトリをローカルにクローンします。
  2. ローカルサーバーを起動して `index.html` を表示します（下の README を参照）。
  3. 別ブランチを作ってファイルを編集し、コミット・切り替え・マージの練習をします。

練習メニュー

例 — ブランチ練習

    changemessage
  1. main　 ブランチから新しいブランチを作成してチェックアウト
    2. =======
  2. main ブランチから新しいブランチを作成してチェックアウト
    3. main
  3. ファイルを編集してコミット
  4. main に戻って変更をマージ
  5. 不要ならブランチを削除
# 例
# git switch -c feature/my-change
# edit files
# git add .
# git commit -m "Add my change"
# git switch main
# git merge feature/my-change
# git branch -d feature/my-change
